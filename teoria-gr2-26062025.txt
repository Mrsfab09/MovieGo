(E)-opisowe
Opisz technologie wykorzystywane przez system wordpress (miedzy 100 a 150 słów)
PHP – darmowy skryptowy język programowania służący do pisania: sklepów internetowych, aplikacji webowych, systemów CMS itp. Jest to technologia back-end czyli działa po stronie serwera. Pliki mają rozszerzenie .php. Jest to jeden ze składników serwera Apache.

MySQL – darmowy silnik relacyjnych baz danych utrzymywany przez firmę Oracle. Narzędziem wspomagającym pracę z bazami MySQL jest PHPMyAdmin. Bazy MySQL projektujemy do stron pisanych w języku PHP.

HTML – jest to język znaczników do programowania szkieletów stron www. Jest to technologia front-end czyli działa po stronie przeglądarki. Do zinterpretowania kodu HTML potrzebujemy dowolnej przeglądarki internetowej. W HTMLu deklarujemy takie elementy jak: nagłówki, teksty, obrazy, listy itp.

CSS – są to kaskadowe arkusze stylów służące do stylizacji kodu HTML. Stylizacji podlegają miedzy innymi: czcionki, obrazy, odstępy, marginesy, kolory itp. Został stworzone w celu odseparowania struktury strony od warstwy wyglądu. Jest to technologia front-end
